# Eazie — Super Content Creator Claude-Ready Project Pack

This pack is designed to help Claude build a premium dual-mode website for **Eazie**:
1. **Bio Mode** — a high-converting link-in-bio experience
2. **Full Experience Mode** — a cinematic artist/creator website

## Included files
- `CLAUDE.md` — master build brief and execution instructions
- `site-architecture.md` — information architecture and page/section logic
- `component-map.md` — recommended components and their behavior
- `design-direction.md` — visual, UX, and motion direction
- `content-guide.md` — structured content model and editable config fields
- `copy-deck.md` — starter copy and CTA language
- `implementation-notes.md` — technical guidance for Next.js and Spotify embeds

## Core idea
The website should begin as an elevated mobile-first link hub, then transform into an immersive full website when the user selects **Enter Full Experience**.

## Primary goals
- Make Eazie look premium, memorable, and intentional
- Keep the landing state optimized for social traffic
- Promote music directly through embedded Spotify players
- Create a scalable structure that can evolve with his brand

## Suggested stack
- Next.js
- Tailwind CSS
- Framer Motion
- Spotify embed iframes
- Modular content config for easy updates

## Recommended workflow
1. Give Claude this full folder or upload the markdown files
2. Ask Claude to inspect the reference link-in-bio:
   - https://popl.co/card/3fbOVBuC/3

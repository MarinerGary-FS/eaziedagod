# Visual Style System — Eazie

## Core Identity

Eazie is not just a creator or artist.

He is a:
- digital world builder
- cultural presence
- brand ecosystem (H@VEN)

The experience should feel like:
"You didn’t land on a page. You entered a space."

Tone:
- cinematic
- confident
- controlled
- immersive
- premium

---

## Color System

### Base
- #0A0A0A (primary background)
- #121212 (section background)
- #1A1A1A (card surfaces)

### Text
- #FFFFFF (primary)
- #B3B3B3 (secondary)
- #6B6B6B (subtle UI)

### Accent (Primary)
- #6C5CE7 (electric indigo)
- glow: rgba(108, 92, 231, 0.4)

### Optional Secondary Accent (use sparingly)
- #00F0FF (neon cyan)
OR
- #FF3D81 (soft neon pink)

---

## Typography

### Headline Font
- Space Grotesk (preferred)
- or similar modern display font

### Body Font
- Inter or DM Sans

### Rules
- strong hierarchy
- large hero text
- clean readable body
- no clutter
- no novelty fonts

---

## Layout & UI Style

### Cards
- background: #1A1A1A
- subtle border (low opacity)
- radius: 12–16px
- soft shadow

### Glass (optional)
- light blur
- low opacity
- used sparingly

### Spacing
- generous
- consistent
- section-based rhythm

---

## Motion System

Use Framer Motion.

### Principles
- smooth
- intentional
- minimal but impactful

### Bio → Full Experience
- fade out bio elements
- expand layout
- stagger in sections
- deepen background

### Section Reveals
- opacity: 0 → 1
- translateY: 20px → 0
- duration: ~0.4–0.6s

### Hover
- scale: 1.02–1.04
- glow effect
- elevation

---

## Background Treatment

- dark gradient base
- subtle animated glow or lighting
- optional noise/grain texture
- no flat/static backgrounds

---

## Music Section Styling

Spotify embeds MUST be wrapped in styled cards.

Each card includes:
- label (Featured Album / Latest Drop)
- title
- embed
- spacing + padding
- subtle glow or border

Layout:
- desktop: side-by-side
- mobile: stacked

---

## Mobile UX (Priority)

- large tap targets
- stacked layout
- fast loading
- minimal clutter

Include:
- sticky bottom action bar:
  - Listen
  - Watch
  - H@VEN
  - Toggle Mode

---

## Premium Experience Rules

The site MUST feel:
- intentional
- elevated
- cinematic
- modern

Avoid:
- template feel
- clutter
- weak spacing
- generic UI

---

## Final Standard

If the result feels like:
- a basic linktree
- a generic template
- a developer default layout

It is NOT acceptable.

This must feel like a premium artist experience.
